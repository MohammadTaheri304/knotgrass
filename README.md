## Knotgrass

### Prerequisites to develop and test project
- JDK 11 (JavaSE 11)
- Maven 3.6+

### How to build?
Check prerequisites, then run:
```
mvn package
```

### How to test?
```
mvn clean test
```

### Javadoc
To generate javadoc, run:
```
mvn javadoc:javadoc
```
